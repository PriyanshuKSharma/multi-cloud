% ---------------- ABSTRACT ----------------
\chapter*{Abstract}
\thispagestyle{plain}
\addcontentsline{toc}{chapter}{Abstract}

The transition from traditional on-premise infrastructure to cloud computing has significantly transformed how organizations design, deploy, and scale applications by offering enhanced flexibility, scalability, and cost efficiency. However, reliance on a single cloud provider introduces challenges such as vendor lock-in, increased costs, limited interoperability, and risks of service outages. These limitations have driven the adoption of multi-cloud architectures, where workloads are distributed across multiple platforms to improve resilience and performance. Despite its benefits, multi-cloud environments introduce complexities in infrastructure management, monitoring, security, and operational consistency due to differences in provider-specific interfaces and APIs. This project presents Democratizing AWS Cloud Operations: A Unified Orchestration Approach to Standardized Infrastructure Management (with the Nebula platform), a unified orchestration platform designed to simplify the management of resources across Amazon Web Services (AWS), Microsoft Azure, and Google Cloud Platform (GCP). Nebula provides a centralized control plane that abstracts cloud-specific complexities, enabling seamless provisioning and management through Infrastructure as Code (IaC) using Terraform. The platform integrates an asynchronous execution framework powered by FastAPI and Celery for efficient and responsive operations. A key feature is an AI-driven Troubleshooting Copilot, which analyzes execution logs to provide accurate remediation suggestions, reducing debugging time and operational effort. Additionally, Nebula includes a Security Vault for secure credential management and a real-time inventory syncing system to maintain an updated view of resources across clouds. The system is designed to be scalable, modular, and extensible, making it suitable for both enterprise and research applications. Overall, Nebula enhances operational efficiency and reduces complexity by enabling users to provision, monitor, and troubleshoot multi-cloud resources through a unified interface, eliminating the need to navigate multiple cloud consoles.
